# Documentation

Personal documentation repository containing notes, cheat sheets, and guides
for various technologies and topics.

## Contents

- **Course Notes** - Security training materials (TryHackMe, Hack The Box,
  PortSwigger, Trailhead), prompt engineering, and other course notes
- **Infrastructure** - AWS services, security best practices, and deployment
  guides
- **Operating Systems** - GNU/Linux distributions, desktop configurations,
  Windows (PowerShell, WSL)
- **Tools** - CLI tools, Docker, Git, and other development utilities
- **AI/ML Concepts** - Machine learning and AI terminology reference

## Structure

Documentation is organized by topic and source, making it easy to find
relevant information quickly. Most files are markdown-based and follow
consistent formatting standards.
